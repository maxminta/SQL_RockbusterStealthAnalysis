# SQL_RockbusterStealthAnalysis
## Project Summary 
Rockbuster Stealth LLC is transitioning its traditional movie rental business into the digital age by launching a new online streaming platform. By capitalizing on its vast catalog of existing movie licenses, the company aims to establish a strong market position alongside industry leaders like Netflix and Amazon Prime. This analysis provides the data-driven insights necessary to guide the service’s launch strategy and ensure long-term commercial growth.
## Key Questions 
● Which movies contributed the most/least to revenue gain?

● What was the average rental duration for all videos?

● Which countries are Rockbuster customers based in?

● Where are customers with a high l ifetime value based?

● Do sales figures vary between geographic regions?
## Systems Used
pgAdmin4: For managing PostgreSQL databases.

PostgreSQL: Relational database management system (RDBMS) used to store and query Rockbuster's data.
